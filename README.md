# Rocket.Chat.Cordova
Rocket.Chat Cross-Platform Mobile Application via Cordova

# How to run
#### Requirements
 * npm
 * nodejs 0.12

#### Install dependencies
```shell
npm install cordova coffee-script -g
```

#### Build
```shell
cordova build ios
```
or
```shell
cordova build android
```

#### Run on simulator
```shell
cordova simulate ios
```
or
```shell
cordova simulate android
```
